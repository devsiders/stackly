## ¿Qué es Stackly?

Stackly es un directorio curado de herramientas esenciales para desarrolladores y diseñadores web. En lugar de perder tiempo buscando librerías, componentes, iconos o fuentes cada vez que empiezas un proyecto — Stackly los centraliza en un solo lugar, organizados por categoría y listos para explorar.

## Categorías

| Categoría | Descripción |
| --- | --- |
| ⚙️ **Libraries** | Librerías y frameworks para frontend y backend |
| 🎨 **UI** | Componentes y kits de interfaz de usuario listos para usar |
| 🧩 **Icons** | Colecciones de iconos SVG e icon fonts |
| ✍️ **Fonts** | Fuentes tipográficas para proyectos web |
| 🧰 **Tools** | Herramientas útiles para desarrolladores y diseñadores |
| 🟣 **No Code** | Plataformas para crear productos sin escribir código |

## Stack tecnológico

- **React** + **TypeScript** — UI robusta y tipada
- **Tailwind CSS** — estilos utilitarios
- **Vite** — build y desarrollo rápido

## Ejecución local

```bash
git clone https://github.com/devsiders/stackly.git
cd stackly
npm install
npm run dev
```
